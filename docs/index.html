Installation

Get started with SDC Boost in just a few steps

1

From Chrome Web Store

Visit the Chrome Web Store and search for "SDC Boost" or click "Add to Chrome" to install the extension directly.

2

Manual Installation (Development)

Clone the repository, install dependencies with bun install, build with bun run build, and load the extension from .output/chrome-mv3 directory in Chrome's extension manager.

3

Configure Your Modules

Click the SDC Boost icon in your browser toolbar or use the navbar boost button to open the module control panel. Enable and configure the modules you want to use.
